# Getting Support

Welcome to Playground! Here's how to get help:

## Documentation

- **Getting Started**: See `README.md` for installation and quick start
- **Development**: See `docs/DEVELOPMENT.md` for local setup
- **Architecture**: See `docs/ARCHITECTURE.md` for system design
- **Contributing**: See `docs/CONTRIBUTING.md` for contribution guidelines

## Issues

- **Questions**: Open a [question issue](../../issues/new?template=question.md)
- **Bugs**: Open a [bug report](../../issues/new?template=bug_report.md)
- **Features**: Open a [feature request](../../issues/new?template=feature_request.md)

## Before Opening an Issue

1. Search [existing issues](../../issues) to avoid duplicates
2. Check the documentation for answers
3. Include reproduction steps and environment details for bugs

## Response Times

We aim to respond to issues within:
- **Security issues**: 48 hours (email contact@playground.ai)
- **Bugs**: 1 week
- **Questions**: 1-2 weeks
- **Feature requests**: Reviewed during planning cycles
